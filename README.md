# event_tree
MS excel event tree developed for UQ masters of fire safety engineering program.

How to use:
-   Download event_tree_final.xlsm file.
-   The inputs tab includes input probabilities for each parameter in the tables on the left. Other inputs such as number of apartments and corridor length are incldued in the 'Other inputs' column.
-   The buttons on the right hand side allow for users to toggle certain parameters on or off.
-   The 'Outputs' column outputs the average individual risk and sum of risk based on the selected inputs.
